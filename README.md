# KMS Putusan Pengadilan Narkotika
Knowledge Management System berbasis Java (MVC) untuk mengelola data
putusan pengadilan pidana narkotika.

# Deskripsi Proyek
Aplikasi ini mengelola data putusan pengadilan narkotika (minimal 50 data)
menggunakan arsitektur MVC (Model-View-Controller) dengan fitur CRUD,
pencarian, filter, dan statistik.
